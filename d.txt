这天玩个叫沙包的游戏，其实更有意思的是它的游戏场地，场地是由一个n*n的方阵，一个人开始站在
（1,1），然后向右跑，到（1,2），（1,3）....（1,n），到了边界就向右转，继续跑，（2，n）...  其实就是在起点（1,1）向右跑，一直遇
到边界或者遇到已经走过的方格就右转继续跑，直到把全部方格都跑过一遍（且每个方格只到达一次）。具体样式见样例输出。

Input
输入的第一行是一个整数T（T<=10），表示测试数据的组数：
每组数据包含一个整数n。（1<=n<=100）

Output
每组数据输出一个n*n的方阵，即跑过的游戏场地。体会体会。代码自己去敲。


Sample Input
3
2
3
4
Sample Output
1 2
4 3
1 2 3
8 9 4
7 6 5
1 2 3 4
12 13 14 5
11 16 15 6
10 9 8 7




#include<stdio.h>
#include<string.h>
int x[100][100];
int main()
{
    int a,b,c,d,n,t;
    scanf("%d",&t);
    while(t--)
    {
        scanf("%d",&n);
        memset(x,0,sizeof(x));//memset()初始化数组，包含在string.h
        int en=0;
        a=1;b=0;
        while(en<n*n)
        {
            while(b+1<=n&&x[a][b+1]==0)
                x[a][++b]=++en;
            while(a+1<=n&&x[a+1][b]==0)
                x[++a][b]=++en;
            while(b-1>=1&&x[a][b-1]==0)
                x[a][--b]=++en;
            while(a-1>=1&&x[a-1][b]==0)
                x[--a][b]=++en;
        }
        for(c=1;c<=n;c++)//由1开始
        {
            for(d=1;d<=n;d++)//由1开始
            {
                printf("%d%c",x[c][d],d==n?'\n':' ');
            }
        }
    }
    return 0;
}
